That's What She Said is a plugin that lives discreetly in your WordPress dashboard inspiring you with quotes by notable women.

== Description ==

A simple mash-up of Matt Mullenwheg's original plugin Hello Dolly with Jessamin Smith's Talkbackbot featuring quotes by notable women. That's What She Said is a feminist plugin that lives discreetly in your WordPress dashboard inspiring you with quotes by women on a variety of topics such as work, family, race, discrimination, violence, men, science, peace and love. You can submit notable quotes below and they will be added to the quotes.txt file that accompanies the plugin.

This work is based on two other works

Jessamyn Smith's Talkback bot
https://github.com/jessamynsmith/talkbackbot/

Matt Mullenwheg's Hello Dolly WordPress Plugin
http://wordpress.org/extend/plugins/hello-dolly/

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
